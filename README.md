# RFID-Attendance

RFID Attendence System using arduino uno
 - Arduino Uno
 - Display 20x2
 - Red & Green LED
 - Potentiometer 
 - Buzzer
 - SD card reader
 - RTC Clock module
 - RFID card and Reader
